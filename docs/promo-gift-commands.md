# Промокоды подарочных сертификатов

| Команда / действие | Что делает |
|--------------------|------------|
| `/certificates` | Открывает раздел «Подарочные сертификаты» |
| Кнопка меню «Подарочные сертификаты 🎁» | То же, что `/certificates` |
| Кнопка «Сгенерировать индивидуальный промокод» | Запрашивает выдачу персонального промокода (нужен номер телефона; один код на акцию) |
| `/revokepromo NR-…` | *(админ)* Отключает промокод **только по коду** |
| `/userpromo <telegram_user_id>` или `/userpromo NR-…` | *(админ)* Код и статус; можно искать по коду |
