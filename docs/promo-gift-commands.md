# Промокод клуба (раздел «Специальные предложения»)

| Команда / действие | Что делает |
|--------------------|------------|
| `/offers` или кнопка «Специальные предложения 💝» | Открывает акции клуба и блок персонального промокода (кнопка или готовый текст) |
| Кнопка «Сгенерировать индивидуальный промокод» | Выдаёт персональный промокод (нужен номер телефона для участников клуба; один код на акцию) |
| `/certificates` / «Подарочные сертификаты 🎁» | Только текст про подарочные сертификаты в кассе — **без промокода** |
| `/revokepromo NR-…` | *(админ)* Отключает промокод **только по коду** |
| `/reissuepromo <id>` или `/reissuepromo NR-…` | *(админ)* Новый активный промокод (старый код перестаёт действовать) |
| `/userpromo <id>` или `/userpromo NR-…` | *(админ)* Код и статус |
