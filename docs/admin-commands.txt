Обзор команд администратора (welcome-bot)
==========================================

Доступ есть только у Telegram user_id из переменной ADMIN_IDS в .env
(несколько id через запятую). Остальные пользователи эти команды
отправляют — бот молча не отвечает.

Статистика и данные
-------------------
/stats
  Сколько пользователей в базе и последние 5 регистраций.

/export
  Скачать CSV со всеми контактами (пользователи бота).

/reviews
  Последние 10 отзывов в чате (оценка, проект, текст, email, дата).

/exportreviews
  Файл reviews.csv — все отзывы из базы.

QR и локации
------------
/qr
  Картинка QR-кода на старт бота (для печати). Нужен пакет qrcode[pil].

/qrzone
  Без аргументов — список локаций и подсказка по использованию.
/qrzone <номер>
  QR на мини-приложение карты для зоны (номера из списка в ответе бота).

Контент (медиа в подписи к сообщению с командой)
-------------------------------------------------
/setphoto
  Фото для раздела «Анонсы» (подпись: команда /setphoto).

/setgif
  GIF для розыгрыша (подпись: /setgif).

/setmainphoto
  Фото главного меню.

/setexhibitionphoto
  Фото раздела выставки.

/setcertphoto
  Фото раздела сертификатов.

/setaboutphoto
  Фото блока «О RAZMAN production».

/clearaboutphoto
  Убрать фото из блока «О RAZMAN production».

Рассылка
--------
/broadcast
  Показать подсказку по формату.

/broadcast Текст сообщения
  Текст всем пользователям из базы.

  Либо прикрепить фото или GIF и в подписи указать /broadcast и текст
  (подпись уйдёт как caption).

Подарочные промокоды (индивидуальный код в разделе сертификатов)
-----------------------------------------------------------------
/revokepromo <telegram_user_id>
  Отключить выданный пользователю промокод (после этого код недействителен).
  Повторно выдать промокод по этой же акции этому пользователю нельзя — запись в базе остаётся.

/userpromo <telegram_user_id>
  Посмотреть код и статус (активен / отключён).

Текст сообщения при выдаче (скидка 10%, срок и формулировки) — в bot.py, правки через деплой.

Автоматическая дата окончания: после последнего дня GIFT_PROMO_VALID_UNTIL (Europe/Minsk)
кнопка промокода скрывается, новые коды не выдаются, старые сообщения с кнопкой при нажатии
получают текст «срок истёк».

Прочее
------
Ошибки бота дублируются в личку: жёстко заданный id + все из ADMIN_IDS
(см. error_handler в bot.py) — это не команды, только уведомления.

Команды без проверки админа (/start, /menu, /review, /map и т.д.)
доступны обычным пользователям — в этот файл не включены.
