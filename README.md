# Welcome Bot

Telegram-бот для сбора контактов через QR-код с автоматической выдачей ссылки в группу.

## Быстрый старт

### 1. Создай бота
Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → скопируй токен.

### 2. Настрой окружение
```bash
cp .env.example .env
```
Открой `.env` и заполни:
- `BOT_TOKEN` — токен от BotFather
- `GROUP_INVITE_LINK` — ссылка вида `https://t.me/+xxxx` (из настроек группы)
- `ADMIN_IDS` — твой Telegram ID (узнай у [@userinfobot](https://t.me/userinfobot))

### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

### 4. Запусти бота
```bash
python bot.py
```

### 5. Получи QR-код
Напиши боту команду `/qr` — он пришлёт картинку с QR-кодом для печати.

## Команды администратора

Доступ только для id из `ADMIN_IDS`. Фото и GIF: отправь файл **с командой в подписи** к этому сообщению.

| Команда | Описание |
|---------|----------|
| `/qr` | QR-код на бота для печати |
| `/qrzone` | QR по зонам карты (`/qrzone` — список, `/qrzone 1` и т.д.) |
| `/stats` | Статистика: число пользователей и последние 5 |
| `/export` | CSV со всеми контактами |
| `/reviews` | Последние 10 отзывов в чате |
| `/exportreviews` | Файл `reviews.csv` со всеми отзывами |
| `/broadcast` | Рассылка всем пользователям (текст или фото/GIF + подпись с `/broadcast`) |
| `/setphoto` | Фото раздела «Анонсы» |
| `/setgif` | GIF для розыгрыша |
| `/setmainphoto` | Фото главного меню |
| `/setexhibitionphoto` | Фото выставки |
| `/setcertphoto` | Фото сертификатов |
| `/setaboutphoto` | Фото блока «О RAZMAN production» |
| `/clearaboutphoto` | Убрать фото из блока «О RAZMAN production» |

## Как работает бот

1. Пользователь сканирует QR-код → открывается бот
2. Бот сохраняет имя, username, дату в `users.db`
3. Предлагает поделиться номером телефона (опционально)
4. Отправляет кнопку со ссылкой в группу
